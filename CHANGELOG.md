# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [1.0.0] - 2026-07-01
### Added
- Initial v1.0.0 Release.
- Extracted `@intelligate/core` engine.
- Implemented Clean Architecture dependency graph.
- Created `@intelligate/sdk` with Transport, Auth, and Business Resources.
- Added AI Provider adapters (Groq, Gemini, HuggingFace).
- Added infrastructure adapters (Redis, MongoDB).
- Implemented dynamic cache mapping and rate limits.
- Built interactive `examples/basic` demo application.
- Comprehensive documentation suite.
